# stock-analyze スキル

日本株分析ツール（`/Users/fujie/code/playground/stock-price-analyze/`）を使って、テクニカル分析・シグナルスキャン・バックテストを実行する。

## 実行環境

- **作業ディレクトリ**: `/Users/fujie/code/playground/stock-price-analyze/`
- **Python**: `.venv/bin/python`（uvで管理）
- **実行形式**: `cd /Users/fujie/code/playground/stock-price-analyze && .venv/bin/python main.py <コマンド>`

---

## コマンド選択

ユーザーの意図からコマンドを選ぶ：

| ユーザーの意図 | コマンド |
|---|---|
| 特定銘柄を分析したい | `analyze <ticker>` |
| ニュースや市況感も含めて分析したい | `analyze <ticker> --with-sentiment` |
| シグナルが出ている銘柄を探したい | `scan` |
| 全銘柄のレポートを出したい | `scan --all` |
| 過去のパフォーマンスを検証したい | `backtest <ticker>` |

## 手順

### 1. 意図を解釈する

ユーザーの発言から以下を特定する：
- **銘柄**: 名前 or ティッカー（複数可）
- **操作**: 分析 / スキャン / バックテスト
- **オプション**: センチメント分析が必要か

不明な場合は「ソニーを分析しますか？それともスキャンですか？」のように一言確認する。

### 2. コマンドを実行する

```bash
cd /Users/fujie/code/playground/stock-price-analyze && .venv/bin/python main.py <コマンド>
```

タイムアウトの目安：
- `analyze`: 60秒
- `analyze --with-sentiment`: 90秒
- `scan`: 300秒（全銘柄分）
- `backtest`: 60秒

### 3. 結果を表示する

- `analyze` / `backtest`: 生成されたレポートの**要点を日本語でまとめる**（全文は長いので不要）
  - テクニカル要点、トレードプラン、主要価格レベルを抜粋する
- `scan`: シグナルが出た銘柄の一覧と各シグナルの内容を表示する
- レポートの保存先（`reports/` 以下）をユーザーに伝える
