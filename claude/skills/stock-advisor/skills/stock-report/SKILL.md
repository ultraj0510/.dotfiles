## 時価の取得優先順位（必須）

1. **portfolio.yaml の `current_price`（SBI同期）を正とする**
2. SBIデータがない場合のみ signal_engine.json の価格を参照
3. 両方ない場合は取得単価で代替しその旨を注記

## 分析原則

- **断定的に書く**: 数量を明示（「100株単位で追加買い」）
- **単元株を厳守**: 数量は必ず100株単位。ETF等は保有数量確認
- **ラギング指標より直近価格を優先**
- **信用期限は絶対条件**: 残日数 < 60日の信用ポジションは必ず言及
- **数値で語る**: 「高い/低い」でなく具体的な数値で
- **見送りも明示**: 動かない銘柄も「見送り」と書く

## マクロ環境の読み方

| 指標 | 閾値 | 日本株への影響 |
|------|------|--------------|
| VIX | >= 25 | リスクオフ。BUY保留、HOLD/SELL優先 |
| VIX | <= 13 | 過信（楽観過多）。高値掴み注意 |
| S&P500前日 | <= -1.5% | 日本株寄付き売り圧力。SELL/HOLD |
| S&P500前日 | >= +1.0% | 寄付き買い追い風。BUY後押し |
| USD/JPY | >= 155円 | 円安。輸出株（任天堂・IHI等）に追い風 |
| USD/JPY | <= 140円 | 円高。輸出株逆風、内需株有利 |
| 米10年債 | >= 4.5% | 高金利継続。成長株バリュエーション圧迫 |

## アナリスト目標株価の読み方

| 条件 | 解釈 |
|------|------|
| 現在値がアナリスト目標の **75%以下**（乖離+25%超） | 割安シグナル。BUY/HOLD後押し |
| 現在値がアナリスト目標の **115%以上**（乖離-15%超） | 割高シグナル。SELL/利確検討 |
| アナリスト「買い/強気買い」+ テクニカル強気 | コンビクション高い BUY |
| アナリスト「売り」+ RSI > 70 | 複合SELL |
| 目標株価レンジ 高値÷低値 > 3x | アナリスト見解分かれる。過信禁止 |
| アナリスト人数 < 3人 | カバレッジ薄。目標株価の信頼性低 |
