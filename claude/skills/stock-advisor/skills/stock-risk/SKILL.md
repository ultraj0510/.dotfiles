# stock-risk — 口座・信用リスク管理

## 口座ルール

- **単元株制度**: 標準売買単位は **100株**。数量指示は必ず100株単位
- ETF（1328.T等）や一部銘柄は単元株数が異なるため保有数量確認
- **数量は100株単位**: 追加買い・一部売却とも100株単位

## 信用取引ルール

- **信用期限**: 一般信用は無期限もあり。制度信用は期限管理必須
- **expiry_date 未設定**: 必ずしも問題ではない（一般信用の可能性あり）
- **残日数 < 60日**: ロールオーバー or SELL 緊急検討。必ずレポートで言及
- **追証判定**: 口座全体の委託保証金率で判断（最低保証金維持率20%）
- margin_ratio < 20% → 追証リスク

## 信用取引コスト
- 買建: 年率 2.8%
- 売建: 年率 1.2%
- 期限: 最大180日

## レポートでの信用期限アラート形式

```
## 信用期限アラート
- **TICKER**: open_date → expiry_date（残り X 日）
```
